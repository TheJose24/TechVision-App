NUESTRO PROCESO EN EL DISEÑO DE PÁGINAS WEB

01

DEFINICIÓN DEL PROBLEMA Y OBJETIVOS

  • Definir o entender los resultados que el cliente busca.
  • Con el cliente discutimos su viabilidad y decidimos cuál es el camino que tomaremos con la página web.
  • Si no los tiene muy claros, le ayudamos a definirlos de manera directa, concreta y medible.
  • Revisión del Brief.
02

EXPERIENCIA DE USUARIO (UX)

  • Llega el momento de definir, junto con el cliente, cuál será el contenido.
  • Para cada página anotamos un pequeño listado de las cosas que debería incluir.
  • Cuando el mapa de contenidos está definido, es el turno del cliente para redactar los textos o, en el mejor de los casos, contratarnos para ello.
03

PROPUESTA DE DISEÑO

  • Trabajamos un diseño gráfico personalizado y único, ceñido a los gustos del cliente y/o sus socios.
  • Creamos una experiencia visual original y que comunique los valores de la marca e identidad corporativa definida.
04

PROGRAMACIÓN

  • Maquetación
  • Utilizamos tecnologías como PHP, CSS, HTML5, JavaScript y metodologías como SCRUM.
  • Análisis y Activación.
05

ENTREGA

  • Fase de prueba.
  • Fase de aprobación.
  • Backup y entrega de los archivos.
